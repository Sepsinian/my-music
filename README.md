# 前言, vue-music

## 技术栈

> MVVM框架Vue2.0 + 状态管理 Vuex +前端路由 Vue Router +构建工具 webpack2.0 + ES6 + fetch + sass + flex + svg + Vuex + axios,jsonp +移动端滚动库：better-scroll



##  已经实现的功能

- [x] 图片懒加载
- [x] 历史搜索
- [x] 音乐收藏
- [x] 音乐暂停
- [x] 音乐快进
- [x] 自动播放下一曲
- [x] 下一曲
- [x] 歌手
- [x] 搜索
- [x] loading
- [x] 播放详情页
- [x] 滚动歌词
- [x] 音乐进度的拖拽

#### 注意：由于涉及大量的 ES6等新属性，node 需要 6.0 以上版本

> Vue.js 打造属于自己的音乐App

### 安装与运行

> cd vue-music

> npm install

> npm run dev //服务端运行 访问 http://localhost:8080

> npm run build 项目打包

> node  myserver.js也可以运行 访问 http://localhost:9000
